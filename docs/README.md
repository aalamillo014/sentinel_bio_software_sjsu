# Documentation

This folder contains all supporting documentation for the Sentinel Bio+ flight software system.

## Contents

- Software_Architecture.md  
  High-level overview of the software design and application structure.

- System_Flow.md  
  End-to-end data flow showing how applications interact within the system.

- Packet_Format.md  
  Definition of science and health telemetry packet structures.

## Purpose

The documentation in this folder supports understanding, development, and presentation of the Sentinel Bio+ software architecture. It provides clear descriptions of system design, data flow, and subsystem interactions.

## Notes

These documents are intended for:
- Team members
- Project leads
- Reviewers and instructors

They complement the source code by explaining how the system operates at a high level.
